# EstudoPC
Repositório apenas para guardar testes ou códigos de estudo relacionados a códigos polares

## Comando úteis:

- Para criar o ambiente virtual
```bash
python3 -m venv .venv
```

- Para "ativar" o ambiente virtual
```bash
source .venv/bin/activate
```

- Baixando Komm
```bash
pip install komm
```
